# Education Center Telegram Bot

A Java backend Telegram bot for an education center where users can browse courses, view pricing and groups, and submit applications.

## Features
- Browse available courses
- View prices and group information
- Submit applications
- Admin authentication with password
- Review and update application statuses
- Create course groups

## Tech Stack
- Java
- JDBC
- PostgreSQL
- Maven
- BCrypt
- TelegramBots API

## Project Structure
- handlers
- services
- repositories
- session/state management

## My Role
